## üî¨ Research Quality

Each Tier 1 technique includes:

### Academic Sources (Required)
- ‚úÖ **Primary source** with author, year, title, journal, URL
- ‚úÖ **Key findings** summary (1-2 sentences)
- ‚úÖ **Effect sizes** when available (Cohen's d, correlation coefficients)
- ‚úÖ **Supporting studies** (2-5 additional citations per technique)

### Implementation Protocols (Required)
- ‚úÖ **Clinical guidelines** from therapy manuals and practitioner guides
- ‚úÖ **Step-by-step process** (setup, core technique, integration)
- ‚úÖ **Contraindications** (when NOT to use)
- ‚úÖ **Best practices** from sports psychology, NLP training, hypnotherapy

### Practitioner Scripts (Required)
- ‚úÖ **Audio script excerpts** (2-4 sentences demonstrating technique)
- ‚úÖ **Specific language patterns** (8-10 phrases per technique)
- ‚úÖ **Delivery notes** (tone, pacing, emphasis, duration)
- ‚úÖ **Milton Model elements** (embedded commands, presuppositions)

### Context & Application (Required)
- ‚úÖ **When to use** (3-6 scenarios)
- ‚úÖ **When NOT to use** (4-6 contraindications)
- ‚úÖ **Target audience** (entrepreneurs, athletes, executives, etc.)
- ‚úÖ **Best for goals** (confidence, focus, energy, etc.)

## üöÄ Integration with AI Generators

### meditation-knowledge-base-v2.ts

The V2 knowledge base provides enhanced selection and integration:

```typescript
// Select optimal techniques for user profile
const techniqueIds = selectOptimalTechniques(userProfile);

// Get full technique details
const techniques = getTechniqueDetails(techniqueIds);

// Generate complete meditation plan
const plan = generateMeditationPlan(userProfile);
```

### Selection Algorithm

Techniques are scored 0-130 points based on:
- **Performance Focus Alignment** (0-50 pts): Higher performanceFocus = better for Myndset
- **Goal Alignment** (0-30 pts): Matches user's primary goal
- **Duration Fit** (0-20 pts): Fits within session length
- **Evidence Preference** (0-20 pts): Skeptical users get strong evidence
- **Audience Match** (0-10 pts): Targets user's performance context

Top 3-5 scoring techniques are selected for each meditation plan.

### Backward Compatibility

V1 meditation plans still work:
- Legacy `MeditationComponent` interface maintained
- Conversion functions provided: `techniqueToLegacyComponent()`
- Old 8 components can be migrated to new database format
